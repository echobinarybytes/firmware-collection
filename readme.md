## File structure
this repo is used to store collection of firmware.
the structure of this repo is :
```c
firm-collection --| .
                --| ..
                --| router // store router firmware
                        --| type 1
                            --| file of router firmware
                        --| type 2
```
文件结构中，`type 1 / type 2`按照厂商分类，`file of router firmare`直接是路由的文件。文件的命名格式待定？
